# hic_tools

1, cool2hic.py

Transfer cool to hic.
```
python3 cool2hic.py -i test.mcool/test.cool -r 10000
```
Options:
  -h, --help            show this help message and exit
  -i <file>, --input=<file>
                        Input cool or mcool file.
  -r <int>, --resolution=<int>
                        Resolution for the output hic file.
